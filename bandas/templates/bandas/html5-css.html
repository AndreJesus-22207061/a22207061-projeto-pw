Seletores CSS

O CSS (Cascading Style Sheets) é uma linguagem utilizada para definir a apresentação de documentos/ficheiros HTML. Os seletores são utilizados para "selecionar" os elementos que queremos estilizar.

  • Seletor: Universal (*)
    Usado:
    Comentário: Seleciona todos os elementos. Não utilizei em nenhuma página.
  • Seletor: Tipo (elemento)
    Usado:
    Comentário: Seleciona todos os elementos de um determinado tipo. Utilizado para estilizar todos os <p> numa página específica. Utilizei nas páginas das Bandas, dos Blogues e do Curso.
  • Seletor: Classe (.classe)
    Usado:
    Comentário: Seleciona todos os elementos de uma determinada classe. Não utilizei em nenhuma página.
  • Seletor: ID (#id)
    Usado:
    Comentário: Seleciona um elemento único com um ID específico. Não utilizei em nenhuma página.
  • Seletor: Atributo ([atributo])
    Usado:
    Comentário: Seleciona elementos com um atributo específico. Utilizado para estilizar links com o atributo "target" numa página específica. Utilizei nas páginas das Bandas, dos Blogues e do Curso.
  • Seletor: Pseudo-classe (:pseudo-classe)
    Usado:
    Comentário: Seleciona elementos num estado específico. Utilizado para estilizar links quando são "hovered" numa página específica. Utilizei nas páginas das Bandas, dos Blogues e do Curso.
  • Seletor: Pseudo-elemento (::pseudo-elemento)
    Usado:
    Comentário: Seleciona e estiliza uma parte específica de um elemento. Não utilizei em nenhuma página.

Combinadores de seletores

  • Combinador: Descendente ( )
    Usado:
    Comentário: Seleciona elementos que são descendentes de um elemento especificado. Utilizado para estilizar <p> dentro de <div> numa página específica. Utilizei nas páginas das Bandas, dos Blogues e do Curso.
  • Combinador: Filho (>)
    Usado:
    Comentário: Seleciona elementos que são filhos diretos de um elemento especificado. Não utilizei em nenhuma página.
  • Combinador: Adjacente (+)
    Usado:
    Comentário: Seleciona elementos que são imediatamente precedidos por um elemento especificado. Não utilizei em nenhuma página.
  • Combinador: General Sibling (~)
    Usado:
    Comentário: Seleciona elementos que são precedidos por um elemento especificado, não necessariamente imediatamente. Não utilizei em nenhuma página.
