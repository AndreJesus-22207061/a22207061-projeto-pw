A Minha Experiência em Programação Web

Sobre Mim

Olá! Sou um jovem apaixonado por tecnologia e inovação. Esta página serve como uma carta de apresentação, onde mostro as minhas habilidades e conhecimentos obtidos na disciplina de programação web, com foco no framework Django.

Tecnologias Utilizadas

Python

Python é uma linguagem de programação poderosa e fácil de aprender. Utilizei Python para a manipulação de dados no backend do projeto. A sua sintaxe simples e a vasta quantidade de bibliotecas tornam o desenvolvimento rápido e eficiente.

Django

Django é um framework web de alto nível que incentiva o desenvolvimento rápido e o design limpo e pragmático. Com Django, construí a estrutura principal do site, configurei a base de dados e implementei funcionalidades complexas de forma segura e escalável.

HTML

HTML (HyperText Markup Language) é a espinha dorsal de qualquer página web. Usei HTML para estruturar o conteúdo e os elementos da página, garantindo uma base sólida e semântica para o site.

CSS

CSS (Cascading Style Sheets) é utilizado para estilizar o HTML, proporcionando um layout visualmente atraente. Apliquei CSS para criar um design responsivo e moderno, ajustando cores, fontes e layouts para melhorar a experiência do utilizador.

JavaScript

JavaScript é uma linguagem de programação essencial para a criação de páginas web dinâmicas e interativas. Utilizei JavaScript para adicionar funcionalidades interativas ao site.

APIs

As APIs (Application Programming Interfaces) permitem a comunicação entre diferentes sistemas e a integração de serviços externos. No projeto, utilizei APIs para integrar a API do IPMA no meu website de meteorologia.
